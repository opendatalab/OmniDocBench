则 $P \left( X = 0 \right) = \frac{C_{5}^{0} C_{4}^{3}}{C_{9}^{3}} = \frac{4}{84} = \frac{1}{21}$, $P \left( X = 1 \right) = \frac{C_{5}^{1} C_{4}^{2}}{C_{9}^{3}} = \frac{30}{84} = \frac{5}{14}$,
$P(X=2) = \frac{C_{5}^{2} C_{4}^{1}}{C_{9}^{3}} = \frac{40}{84} = \frac{10}{21}$, $P(X=3) = \frac{C_{5}^{3} C_{4}^{0}}{C_{9}^{3}} = \frac{10}{84} = \frac{5}{42}$.

∴ 随机变量 $X$ 的分布列为

| $$X$$ | 0 | 1 | 2 | 3 |
|------|---|---|---|---|
| $$P$$ | $$\frac{1}{21}$$ | $$\frac{5}{14}$$ | $$\frac{10}{21}$$ | $$\frac{5}{42}$$ |

随机变量 $X$ 的数学期望 $E \left( X \right) = 0 \times \frac{1}{21} + 1 \times \frac{5}{14} + 2 \times \frac{10}{21} + 3 \times \frac{5}{42} = \frac{5}{3}$.

如图, 在三棱锥 $A - BCD$ 中, $\bigtriangleup ABD$ 为等腰直角三角形, $AB = AD$, $\bigtriangleup BCD$ 为等边三角形.

(1) 证明: $BD \perp AC$;

(2) 若直线 $AC$ 与平面 $ABD$ 所成的角为 $\frac{\pi}{3}$, 点 $E$ 在棱 $AD$ 上, 且 $DE = 2EA$, 求二面角 $E - BC - D$ 的大小.

![$$A$$ $$E$$ $$B$$ $$D$$ 第(20)题图 $$C$$](figures/1.1)

解: (1) 证明: 如图, 取 $BD$ 的中点 $O$, 连接 $OA$, $OC$,

∵ $AB = AD$, ∴ $BD \perp AO$,

∵ $\bigtriangleup BCD$ 为等边三角形, ∴ $BD \perp CO$,

∴ $BD \perp$ 平面 $AOC$,

又 ∵ $AC \subset$ 平面 $AOC$,

∴ $BD \perp AC$.

(2) (解法一) 由 (1) 不难知道, 在平面 $AOC$ 中, 若过 $C$ 作直线 $AO$ 的垂线, 则该垂线亦为平面 $ABD$ 的垂线, 故直线 $AC$ 在平面 $ABD$ 内的射影为直线 $AO$,

∴ $\angle OAC$ 为直线 $AC$ 与平面 $ABD$ 所成的角, 即 $\angle OAC = \frac{\pi}{3}$.
