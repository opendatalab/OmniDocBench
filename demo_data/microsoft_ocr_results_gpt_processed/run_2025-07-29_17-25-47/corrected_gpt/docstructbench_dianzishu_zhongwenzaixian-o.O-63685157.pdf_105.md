# 像嬉皮一样在马赛街头晃荡

法国给我的感觉一直很高冷，法国人都是些不太爱搭理人的讲究人，他们讨厌被打扰。

马赛是一个港口城市，我们下船打车，车是本土车型，很新，没见过的那种车型，车是6座的，顶棚很高，座位很灵活，但体积不大，同行的5个人都能坐进去还放了行李。我特别希望咱们中国也有这样的出租车，我跟乐队哥们儿去演出时正需要这样的车。司机师傅是一个在我看来长相很具法国典型特点的中年人，如果你看过《纽约我爱你》，里面用醋在纸上画下舒淇的样子的那位画家，就是那种长相。他们的high点很奇怪，但总感觉他们的微笑和低沉的语气里有另外一种习惯，不爱与你分享。这也许就是大多数法国人给人感觉高冷的原因，其实他们自己活得可自在了。

下车就是马赛的老码头，都是游艇，桅杆不是很高。路边有些卡通人物在跟你招手，还有一个很好看的街头艺术，他们用重力错乱的视觉效果让你眼前一亮，这种东西在国外一点儿也不稀奇，但空间和重力的错乱真的挺需要想象力的，而且他们想到就去做了，我觉得这些人很可爱，他们在司空见惯里制造着错乱，让人们笑了一下，精神了一点儿。
