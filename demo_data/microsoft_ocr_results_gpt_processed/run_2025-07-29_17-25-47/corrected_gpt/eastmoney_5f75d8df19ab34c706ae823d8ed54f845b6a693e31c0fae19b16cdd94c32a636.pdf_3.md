# ■ 证券投资评级说明

## 股票投资评级说明:

| | 投资建议 | 预测个股相对同期证券市场代表性指数涨幅 |
| --- | --- | --- |
| 1 | 买入 | > 20% |

## 行业投资评级说明:

| | 投资建议 | 行业指数相对同期证券市场代表性指数涨幅 |
| --- | --- | --- |
| 1 | 推荐 | > 10% |
| 2 | 中性 | -10%-10% |
| 3 | 回避 | < -10% |

以报告日后的12个月内,预测个股或行业指数相对于相关证券市场主要指数的涨跌幅为标准。

相关证券市场代表性指数说明:A股市场以沪深300指数为基准;新三板市场以三板成指(针对协议转让标的)或三板做市指数(针对做市转让标的)为基准;香港市场以恒生指数为基准;美国市场以道琼斯指数为基准。
