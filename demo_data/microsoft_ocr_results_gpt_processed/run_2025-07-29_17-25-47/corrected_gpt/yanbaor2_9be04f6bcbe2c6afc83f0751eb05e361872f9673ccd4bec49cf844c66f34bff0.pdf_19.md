# 扬帆远航,绽放多元魅力

陈晓颖女士  
德勤中国税务总监

![](figures/1.1)

“我很喜欢卡尔维诺的一段话--我对任何唾手可得、快速、出自本能、即兴、含混的事务没有信心。我相信缓慢、平和、细水长流的力量、踏实、冷静。我不相信缺乏自律精神、不自我建设和不努力,可以得到个人或集体的解放。”

回顾2020年,突如其来的疫情对原有的工作方式及生活产生了不可忽视的影响,作为德勤税务总监,陈晓颖同样将这次挑战视为自我建设、自我解放的一股东风,乘风破浪。

在去年9月底,陈晓颖作为日本服务小组(JSG)世代领导者里唯一的女性成员受邀主持了全球JSG大会的亚太区会议,这也是全球JSG首次利用线上远程技术举办会议。

面对疫情大环境给以往的工作方式带来的冲击,陈晓颖探寻新的方式来适应:“对外,我们通过推送线上网络直播会议增加客户粘性,待疫情趋于稳定以后,通过圆桌会议的形式去做更多线下的交流;对内,为了充分调动团队成员的主观能动性,在疫情大环境中迎难而上,我提议今年我们组的行动计划由大家群策群力,从新颖的产品到独特的推广方法,制定出凝聚每一位经理巧思的行动计划,并且定期讨论,行之有效地推进落实。”

作为女性领导者,陈晓颖不但在职业舞台上乘风破浪,在公益、生活、家庭舞台上同样扬帆远航。就如她所言“女性的领导风采是多元的,刚柔并济的。”

作为新一届的德勤“智启非凡”大使,从参与“新一千零一夜”项目担任“德勤好声音”,为乡村寄宿制学校学生录制睡前故事,到筹划全国爱心日“良师益友”活动,帮助特殊青少年融入上海生活,陈晓颖从一名志愿者成长为组织者。在公益道路上,陈晓颖特别感谢来自公司的支持,“德勤一直鼓励灵活的工作方式,公司在包括公益项目推广上,也充分考虑了疫情所造成的影响,开展更多线上活动,坚持在保障安全的情况下开展公益项目。”

作为德勤“勤跑”俱乐部的成员,去年10月,陈晓颖发挥自己长跑项目的优势,代表德勤参加黄浦区商委主办的外企高管和国际友人健康跑,展现了德勤人积极向上的风采。

作为两个孩子的母亲,陈晓颖心系团队成员的美好生活,为遇到职业瓶颈的成员解惑,努力成为年轻女同事的榜样。

“正如德国作家赫尔曼·黑塞在书中写道的,面对呼啸而至的时代车轮,我们必须加速奔跑,它可以轻易将每一个落伍的个体远远抛下。我想对刚入职的女性员工说,做奔跑在路上的人,才有机会遇见更好的自己。”

在时代的巨浪中,陈晓颖呼吁德勤的女性同仁甩开绳索,离开避风港,乘着东风,扬帆远航,在更多的舞台绽放多元的女性魅力。
