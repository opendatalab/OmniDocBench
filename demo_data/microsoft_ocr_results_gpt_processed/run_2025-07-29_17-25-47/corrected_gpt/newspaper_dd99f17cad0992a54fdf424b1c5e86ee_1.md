# 浙江省农作物病虫害防治条例1月1日实施

本报讯 国内首部有关农作物病虫害防治工作的地方性法规《浙江省农作物病虫害防治条例》，于2011年1月1日起在该省实施。自此，农作物病虫害防治开始在浙江走上法制化轨道。

浙江地处东南沿海，是各类农作物病虫害频繁发生的一个省份，尤其是近年来，受气候、环境变化等影响，全省病虫害的发生和危害已呈上升趋势。据有关部门统计，全国1640多种农作物病虫害，在浙江大部分都已出现。如何做好各类农作物病虫害的防治，确保农业增效，农民增收，是当前发展现代农业进程中必须重视的一个重大课题。

新条例对浙江农作物病虫害防治的工作主体，监测与预报制度、财政投入机制及农药经营和农药废弃包装物回收制度等各个方面均做了明确的规定。

# 兰州市为农村五保户发放物价补贴

本报讯(记者 吴晓燕)甘肃省兰州市从2010年11月至2011年3月，为城乡低保、农村五保和重点优抚对象发放5个月的临时物价补贴，一次性发放。城市低保补贴标准为每人20元/月，共计100元；农村低保补贴标准为每人10元/月，共计50元；农村五保补贴标准为每人20元/月，共计100元；优抚对象补贴标准为每人20元/月，共计100元。

为稳定物价，兰州市建立了居民消费价格总水平(CPI)上涨与城乡低保，农村五保临时物价补贴联动机制。当兰州市CPI连续三月增长超过3%(含3%)时，于次月自动启动临时物价补贴调节机制。当月度CPI增长低于3%(不含3%)时，于次月自动取消临时物价调节补贴。

# 庆元县农民收入 香菇效益占一半以上

本报讯(记者 徐盘钢)记者从浙江省庆元县人民政府日前在上海举行的新闻发布会上获悉，由庆元县人民政府承办的第八届中国(庆元)香菇文化节于日前在庆元举行。

庆元县素有“中国香菇之乡”、“中国香菇城”之称。全县19万人口，半数以上从事香菇生产及其相关行业，全县农民人均收入和地方财政收入中香菇效益占了50%以上。据统计，2009年全县食用菌总产值达到3.2亿元，共生产各类食用菌达1.25亿袋。

## 灌云县80%农村五保老人实现集中供养

本报讯(潘兴祥 记者 陈兵)记者日前从江苏省灌云县民政部门获悉，截至目前，该县现有的2400多名五保老人中，有1900多名实现了集中供养，农村五保老人集中供养率达80%。

据介绍，2010年以来，灌云县积极开展五保户“关爱工程”项目建设，安排专项资金用于改善敬老院住房条件。目前，全县敬老院基本达到院内环境园林化、居住宾馆化。在做好硬件建设的同时，灌云县民政部门还通过配备专职工作人员，各项经费落实到位等多种措施提高集中供养率，目前，全县八成农村五保户实现了集中供养，生活待遇明显改善。

# 从首创《村规民约》起步

--广西宜州市合寨村“村民自治”30年纪实

## 【核心提示】

合寨村村民创造性地建立的农村群众性自治组织--村民委员会，直接影响了日后中国农村经济体制改革，开辟了中国村民自治的新天地。

本报记者 王家增

30年前，广西宜州市屏南乡合寨村通过“民主选举，民主决策、民主管理，民主监督”，实行“自我管理、自我教育、自我服务”，成为以“村民自治”为标志的中国农村体制变革的先行者。

合寨村的“村民自治”，得到了党和国家的肯定。1982年“村民自治”写进了中国宪法，1997年写入了党的十五大报告，成为了1998年正式实施的《中华人民共和国村民委员会组织法》的具体规定，“村民自治”从此走向全国。

《村规民约》:首开中国“村民自治”先河

合寨村，一个只有在县级地图上才能找到的穷乡僻壤。如果不是30年前有一个人吹响了“集结号”，它或许永远默默无闻，更不可能影响中国农村法制体制建设的进程。

这个人就是蒙宝亮，一位退伍军人。1975年担任合寨大队党支书，一干10年，不但改变了合寨村的命运，而且为中国农村法制建设写下了浓墨重彩的一笔。

上任之初，合寨村偷盗赌博，滥伐山林、打架斗殴等不良风气盛行，4000多人的村子，参赌竟有上千人，各种丑恶现象层出不穷。

合寨村人把各种乱象概括为“六多一少”，即赌博多、盗窃多、乱砍滥伐多，唱山歌多、放浪牛浪马多、搞封建迷信多，管事的人少。

这股歪风邪气刹不住，村民难有宁日。

1979年底的一个晚上，蒙宝亮召集党员，干部开会，决心要找出个办法来。大家你一言我一语纷纷出谋献计。最后，蒙宝亮归纳大家的意见：“我们就订一个《村规民约》，用来管住那些歪风邪气。”

回忆起当时的一幕，今年78岁的蒙宝亮依然记忆犹新：“我的话音一落，蒙成顺一拍大腿，兴奋地说‘这个办法好!’两个小时后，一个具有划时代意义的《村规民约》初稿形成了。”

这是中国农村法制建设具有里程碑意义的一个晚上。

《村规民约》让合寨村首开中国“村民自治”之先河，成为新中国农村体制改革当之无愧的发源地。

## 村民:看重自己手中神圣一票

1984年撤社设乡后，合寨大队成立了村民委员会，合寨村果作屯的党员书焕能以出色的人品和能力当选村委会主任。在他的带领下，村里各项工作大有起色。

事实让村民们深深体会到民主选举的力量。因此，村委会的每一次换届选举，村民们都表现出强烈的民主意识，人人都十分看重自己手中神圣的一票。

1999年9月25日，是合寨村村民委员会的第六届选举日。已离开本村10年的蒙桂北，在得知自己的选民资格被遗漏后，放下手中的工作立即从外地赶回村里，向村民委员会申请选民资格，投上自己神圣的一票。

随着时代变迁，合寨村的投票方式也在与时俱进。

2008年4月11日，合寨村进行第九届换届选举，除通过电话投票外，还开通了网络视频投票。远在深圳打工的村民蒙超赶到网吧，通过视频向记录员说出心仪的候选者，请他代填选票。

就这样，千里之外的蒙超如愿地投出了庄严的一票。

如今，合寨村外出务工人员超过全村总人数的一半，但村民的参选率高达93%。

30年来，村委会领导班子换了九任，每一任都是村民用自己神圣的选票选出。村委会领导班子在财务收支，基础建设、生产发展等关乎全村利益的大事上，也都广泛征求意见，由村民说了算。

随着形势发展，思想活跃，富有开拓精神，见多识广的新型带头人成为了村民们选举的标准。

书向生当选该村第七届村民委员会主任后，带领合寨村建成了桑蚕，甘蔗两大支柱产业；建起了3个食用菌示范基地；涌现一批猪、牛、鹅、兔、鱼专业养殖户。村民们的致富路越走越宽，人均纯收入从1980年的57.83元提高到2009年的4345元；85%以上的农户建起了楼房，汽车、电脑、冰箱等普及农家。

## 村干部:真心实意为群众服务，群众才会拥护

合寨村村民对民主权利的充分运用，莫过于选到了敢于承担责任、无私奉献的带头人。

在首任村官蒙宝亮的言传身教下，勇担责任，乐于奉献在合寨村干部心中扎下了根，由此而建立起来的威信使村干部手中的指挥棒越来越灵验。

30年如一日，合寨村九届村委会领导班子在实施村民自治的过程中，以勇担责任，无私奉献精神严格要求自己，并作为治村法宝的“接力棒”一届届传承，使国家的政策法令得到正确的贯彻执行，《村规民约》成为了“村民意志”，一个欣欣向荣的社会主义新农村因此矗立在世人面前。

合寨村村民创造性地建立的农村群众性自治组织--村民委员会，直接影响了日后中国农村经济体制改革，开辟了中国村民自治的新天地。2010年8月18日，时值合寨村“村民自治”30周年之际，全国人大常委会委员长吴邦国欣然为该村泼墨题词：“中国第一个村民委员会”。与安徽小岗村“包产到户”齐名，并称中国农民两个伟大创举的“村民自治”，在中国农村政治民主建设的突出地位和重要作用再次得到了党和国家的肯定。

本报记者 江娜

发展现代农业，离不开农药，化肥等生产资料的大量使用。农民用药水平的高低，农药使用的安全与否，直接关系到农作物病虫害的防治效果、施药人员身体健康，农产品质量安全和生态环境安全。但由于我国农业生产组织化程度低，农户众多，对农药使用的指导和监管是目前农药管理各环节中难度最大，最薄弱的一环。
