# 全书大归纳

## 核心知识归纳

### 一元一次方程

| 知识点 | 内容 |
| - | - |
| 方程的解的概念 | 使方程左、右两边的值相等的未知数的值就是该方程的解. 提示:将方程的解代入方程,等式成立 |
| 等式的基本性质 | (1)等式两边都加上(或都减去)同一个数或同一个整式,所得结果仍是等式.如果$a=b$,那么$a+c = b+c$, $a-c = b-c$ ; (2)等式两边都乘以(或都除以)同一个数(除数不能为0),所得结果仍是等式. 如果$a = b$, 那么$ac=bc$, $a/b=b/c$ ($c≠0$) |
| 方程的变形规则 | (1)方程两边都加上(或都减去)同一个数或同一个整式,方程的解不变; (2)方程两边都乘以(或都除以)同一个不等于0的数,方程的解不变 |
| 移项 | 将方程中的某些项改变符号后,从方程的一边移到另一边的变形. 注意:移项后要改变符号 |
| 一元一次方程的定义 | 只含有一个未知数,并且含有未知数的式子都是整式,未知数的次数都是1,像这样的方程叫做一元一次方程. 提示:一般式为$ax=b$ ($a,b$为常数,且$a≠0$) |
| 解一元一次方程的步骤 | (1)去分母;(2)去括号;(3)移项;(4)合并同类项;(5)将未知数的系数化为1. 提示:应根据方程特点,灵活选择解题步骤 |
| 列一元一次方程解应用题的几种常见题型及特点 | 类型 | 题中涉及的数量关系及公式 | 等量关系 |
| | 和、差、倍、分问题 | 明确“倍数”关系及“多、少”关系等 |
| | 等积变形问题 | 长方体体积=长×宽×高; 圆柱体体积=$πr^2h$ (其中$h$为高, $r$为底面半径) | 变形前后体积相等 |
| | 行程问题 | 相遇问题 | 路程=速度×时间; 时间=路程÷速度; 速度=路程÷时间 | 快行的距离+慢行的距离=总距离(相向而行) |
| | | 追及问题 | 快行的距离-慢行的距离=距离差(同向而行) |
| | | 水流问题 | 静水速度+水流速度=顺水速度 静水速度-水流速度=逆水速度 (顺水路程=逆水路程) |
| | 调配问题 | 从调配后的数量关系中找等量关系. 注意:调配对象流动的方向和数量 |
| | 比例分配问题 | 全部数量=各份的数量之和. 提示:可把一份数量设为$x$ |
