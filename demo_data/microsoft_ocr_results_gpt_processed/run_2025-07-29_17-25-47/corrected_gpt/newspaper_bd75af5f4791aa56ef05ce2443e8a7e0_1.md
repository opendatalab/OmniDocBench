# 九江县: 产业小区建设成气候

日前,江西九江县新塘乡前进村的王灿帮高兴地说:“在政府的帮助和支持下,我种植白茶近千亩,占全国白茶种植面积的1/3以上,2010年已实现产值300多万元。”据了解,白茶是茶叶中的稀有品种,种植范围有限,价值比一般茶叶高,市场前景好。目前市场上供不应求。

近年来,九江县按照“一村一品”产业发展思路,从资金扶持、政策优惠、金融服务配套等方面大力支持特色产业发展,以蔬菜、林果、苗木等特色产业为重心,大力发展白茶、油茶、葡萄等新兴特色产业。经过几年的发展,特色产业逐步形成规模优势,已初步建立北部九江市近郊蔬菜、南部九江高速公路沿线苗木、西部九江铁路沿线果业等几大特色产业带。截至目前,全县蔬菜面积4.5万亩;苗木面积1.5万亩;果园面积5.5万亩,其中早熟梨面积4.5万亩,形成了区域化布局,规模化经营的特色产业发展格局。

# 雷山县: 农企合作成就规模化发展

近年来,贵州省雷山县大力推广种植新技术、新品种,引导、扶持农民专业合作社健康发展,帮助龙头企业做大做强,引导农民转变种植观念。目前,该县已有近万名农民投身茶叶、山野菜、水果、中药材等产业当中。

参与农民专业合作社或与农业龙头企业合作,是雷山农民融入产业、抱团取暖、发家致富的新“法宝”。为促进农民合作社的健康发展,雷山县工商部门充分发挥职能作用,采取服务措施,突出地域特色和产业优势,积极引导果品、茶叶、山野菜等种植农民专业合作社发展。

该县种植专业合作社已初步形成了技术协作、综合服务、开发应用等多功能形态。望丰村有茶叶基地1500多亩,在茶叶专业合作社的带动下,开办了望丰苗多茶加工厂,加工销售茶叶,年茶叶销售收入近10万元。全村有果品基地1128亩,在村果品协会牵线联系下,果品成熟季节,老板就上门收购,解除了果农的后顾之忧,该村每年果品收入达30多万元。

# 建德市: 夫妻广种白茶 年收入过百万

连日来,浙江建德市乾潭镇罗村白茶种植大户赵小明与聂美英夫妻俩投资50多万元,忙于对50多亩白茶搭建一个个锯架温室大棚,让这些“摇钱树”不受寒潮侵袭。同时,他俩还组织人员加强对150多亩高山白茶园进行除草、施肥等科学培育管理,确保明年白茶增产增收。据了解,现有白茶面积200亩的他们既种白茶又育苗,既深加工又销售,预计2010年收入可过百万元。

自2003年起,夫妻俩除了在家采摘加工白茶进行销售外,又利用自行繁育的白茶苗在家中种植了15亩,通过几年的培育管理,这批茶叶经过采摘加工销售,平均每亩的收入达万元。之后他俩又辗转到气候适宜的杨村桥镇龙源村与长宁村,租赁山地与土地。“接下来,基地至少要繁育500-600万株白茶苗,此外还将对现有种植的200亩普通白茶品种进行逐年更新,发展自行繁育的新品种优质白茶品种,这样不断提高深加工后的产品质量,增强市场竞争能力,扩大当地白茶在外的知名度!”赵小明自信地告诉笔者。

# 利津县北宋镇: “田间课堂”受宠

连日来,山东利津县北宋镇翟家村郭建彬的食用菌大棚里格外热闹,附近村庄许多人来听他讲解食用菌种植技术。“在这里学技术,边听边看,现场示范,脑袋学得快,腰包鼓得也快。”邻村农民吴学亮说。时下,像这样的“田间课堂”在北宋镇还有很多,深受农民百姓的喜爱。

“田间课堂”是北宋镇正在推广的一种新型农民培训模式,它将农村实用技术培训地点设在田间地头,组织种植、加工能手和致富能人现身说法,让村民看得见、摸得着、听得懂、学得快。为使“田间课堂”发挥更大的作用,该镇还将培训工作与产业结构调整紧密结合起来,针对当地不同的季节及技术需求,对农民进行轮流培训。据统计,近一年多以来,北宋镇“田间课堂”已举办32期,累计培训农民达1600余人次。

陕西旱地玉米高产创建出佳绩

# 全膜覆盖双垄沟播玉米亩产超900公斤

2010年9月底,农业部组织玉米专家对陕西省农业厅、西北农林科技大学农学院、榆林市农业局和定边县农业局共同实施的定边县万亩早熟玉米全膜双垄沟播技术高产创建示范片进行了田间现场实测验收。专家组在听取玉米高产创建活动、陕西省测产方法和过程的基础上,对陕西省定边县学庄乡、郝涟镇2个乡镇10086亩玉米高产示范片考察后,随机抽取了30个田块进行了实测,30个点平均产量达到819公斤/亩,其中部分地块亩产超过了900公斤,比当地常规半膜覆盖玉米亩增产300-400公斤,增幅达到一倍以上,亩增收600-800元。

陕北地区年降水量400毫米左右,干旱是制约农业生产的主要因素,但区内光照充足,土地面积广阔,是陕西粮食生产的重要潜力区。为此,该省以高产创建为抓手,在陕北定边县等地开展早地玉米高产创建活动,集成优良品种、合理密植、测土配方施肥、病虫害综合防治等常规技术,建立了7.4万余亩示范区。通过组建农机服务专业队,实行化肥、地膜、农药统购统供等有效措施,保证了各项关键技术落实到位。

早地玉米实现高产得益于集成技术的大力推广,即主推顶凌全膜覆盖、选用耐密品种、测土配方施肥、宽窄行种植、机械复式作业以及统防统治等技术的集成使用。特别是玉米全膜覆盖双垄沟播技术具备四个优点:通过地膜全部覆盖地表,有效抑制地表蒸发,大幅提高水分生产效率;提高地温,使播期提前,解决了当地无霜期短,玉米无法成熟的问题;通过双垄沟播,将5mm以下的无效降雨集中到玉米根系周围,变为有效降雨,提高了天然降雨利用率;顶凌覆膜,有效保住底墒,解决了当地春早玉米出苗难的问题。可以说,玉米全膜覆盖双垄沟播技术的示范推广,突破了干旱、半干旱地区粮食增产的瓶颈,受到了当地农民的欢迎和积极应用,为陕西乃至全国同类地区粮食增产探索了一条科学有效的途径。

为了有效推进玉米高产创建工程,实现集中连片、规模种植,达到高产、高效目标,陕西省拿出500多万元专项资金,用于对陕北玉米全膜双垄沟播技术项目的补贴,其中,定边县采取政策扶持和项目带动的办法,加大对农业的补贴力度,县财政每亩补贴260元,集中用于良种、地膜、农药、机械等环节的补贴,极大地调动了农民采用全膜双垄沟播的积极性。

包括定边县在内的项目县组建了多个农机专业服务队,组织农机农艺技术人员联合开展培训,使农机手掌握了机械操作和农艺要求,实行集中统一规范的覆膜机播作业,确保关键技术落地。这样既解决了农村劳动力不足、覆膜耗时长的问题,提高了工作效率,又保障了覆膜质量,并通过机播实现种植密度的增加。各项目县把项目实施与大面积示范相结合,以点带面,整体推进。立足小田做试验,大田做文章,积极开展早地玉米肥效、密度、品种对比试验,为大田生产示范提供可靠科学依据;通过组织县乡村各级干部、玉米生产大户到示范区进行观摩交流,让群众看有样板,学有榜样,做有典型。

此外,在农资购买上,各项目县改变过去由农户分散购买、价高且质量也难以保证的做法,实行项目区统一供膜、统一供肥、统一供种、统一机械作业。由于购买量大,享受到了优惠价格,降低了生产成本,达到了田间作业的标准化,提高了项目建设标准。同时,为确保各项技术的集成化实施,各项目县都实行了责任追究制度,将任务完成情况纳入年度目标考核范围,并定期对各地项目进展情况进行督促检查。

# 2010年各地种植业亮点纷呈

## 吉林: 粮食生产实现“五增加”

【编者按】

2010年,我国全年粮食总产达10928亿斤,增产312亿斤,实现了半个世纪以来的首次“七连增”。面对严重的自然灾害、农产品市场价格异常波动和复杂的外部环境,取得这样的成就实为来之不易。刚刚结束的全国种植业工作会议上明确提出,2011年要全力以赴争取实现继续增产。本版今日推出“区域亮点”专栏,重点介绍吉林、湖南、广西、四川和陕西五省区在粮食、蔬菜以及推进植保工作等方面的好做法、好经验,供大家学习参考:

2010年对于吉林省粮食生产来说,是极不平凡的一年。多种自然灾害连续发生,多年未遇;粮食生产面临的困难和压力,多年少有。面对困难和挑战,该省坚持把粮食生产作为农业农村工作的重中之重不动摇,全省上下加大政策宣传和落实力度,充分调动农民种粮积极性,通过大力推广增产增效技术和品种,为粮食生产提供科技支撑,深入开展粮食高产创建活动,促进增产技术集成应用,充分利用现代传媒指导农民搞好生产,提高服务时效性,奋力抗灾保根,最大限度减少灾害损失。

通过上述得力措施,粮食生产取得了比预想要好的“五增加”喜人形势:

粮食播种面积增加,尤其是高产粮食作物面积明显增加。全省粮食播种面积达到7194.9万亩,比上年增加267万亩;全省投入春耕生产资金129亿元,比上年增加8.4亿元,化肥投入570万标吨,增加20万标吨,资金和化肥投入均创历史新高;保苗率增加。全省玉米平均保苗率达到97.3%,高于常年3个百分点,高于上年1个百分点,大豆保苗率达到96.4%,高于常年4个百分点;玉米、水稻优质专用品种种植面积增加。玉米、水稻两大主粮作物优质专用品种种植面积达到3972万亩,比上年增加110万亩;粮食单产和总产双增加。全省粮食平均亩产达到421公斤,比上年增加51公斤,粮食总产增加70多亿斤,再创历史新高。

2011年,吉林省将继续抓好增产百亿斤商品粮能力工程项目建设,增产增效技术集成推广,全面开展粮油高产创建活动,计划全省万亩以上规模高产示范田建设数量达到300个,示范面积达到350万亩以上,提高技术集成应用水平。

## 湖南: “五化”推进植保统防统治

湖南是农作物病虫害多发频发、重发省份。为了抓好病虫害防治,推进植保工作,该省积极组建专业化防治组织,大力推进统防统治,努力提升防控能力,着力解决农民一家一户防病治虫难的问题。

近年来,该省通过狠抓“五化”工作,使专业化统防统治成为提高农民组织化、专业化程度的战略切入点,农民形象地称誉为“没有多花钱,请了个田保姆”。

坚持社会化发展,调动各方力量广泛参与。该省通过广泛宣传发动,努力提高专业化统防统治全社会参与度,鼓励农药生产和农资流通企业、各种农业生产合作组织、社会能人,积极投入到专业化统防统治服务中来;坚持项目化促动,整合资源加大扶持力度。为促进专业化统防统治快速发展,2010年,该省从植保专项经费中列支1400万元作为推进工作经费,集中用于36个专业化统防统治项目重点县;坚持规范化管理,提升统防统治服务水平。该省制定出台了统防统治管理办法,在工商部门注册了农作物病虫害专业化统防统治服务组织标志。办法、标志的颁布和使用,既规范了组织服务,又保障了农民权益;坚持市场化运作,良性竞争促进快速发展。湖南省农作物播种面积1亿多亩,按目前防治投入标准计算,仅水稻就可产生专业化统防统治服务费27亿元以上,市场潜力巨大。为此,该省引导扶持涉农企业、社会团体、种田大户、社会能人组建了服务模式多样的专业化服务组织;坚持全程化服务,确保综合防控措施落到实处。

## 广西: 蔬菜产业呈现“五大亮点”

近年来,广西充分利用独特的光热资源、大量的冬闲田以及丰富的冬闲劳动力,大力推动秋冬种蔬菜产业发展,通过实施钱粮双增工程,创新推广良种良法,创建蔬菜标准园,加大农技人员培训,开拓市场抓销售等措施,使广西蔬菜产业呈现“五大亮点”。

南菜北运贡献大。2010年,广西秋冬菜播种面积1300万亩,预计产量1950万吨,调出850多万吨,同比增加80万吨,为华北、长三角、珠三角及港澳市场提供了大量货源。

科学布局规模化。蔬菜产业向优势产区集中,全区有6个市蔬菜播种面积超100万亩,有26个县超20万亩,基本形成了秋冬菜、夏秋反季节菜、中心城市蔬菜、丘陵高山特色蔬菜、食用菌六大商品菜优势产区。
