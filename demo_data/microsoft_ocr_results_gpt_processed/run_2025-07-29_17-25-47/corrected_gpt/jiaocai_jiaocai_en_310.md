## Zhejiang University Theory of Computation, Fall-Winter 2015 Final Exam

1\. (20%) Determine whether the following statements are true or false. If it is true write an O otherwise $\times$ in the bracket before the statement.

(a) ( ) Let $L$ be any regular language. Then the number of equivalence classes respect to language $L$ (i.e. $x \approx_{L} y$, if for all $z \in \Sigma^{*}$, $xz \in L$ iff $yz \in L$) is finite.

(b) ( ) The language $\{a^{n}b^{n}w | w \in \{a, b\}^{*}, n \in \mathbb{N}$, and $|w| = 2n$ is context-free.

(c) ( ) The language $\{M^{\natural} w^{\natural} | M$ accepts $w$ in less than 2016 steps $\}$ is not recursive.

(d) ( ) The language $\{M^{\varsigma_{i}} | M$ is a TM and $L(M)$ is Context-free but $L(M)$ is not regular} is not recursive.

(e) ( ) The language $\{M_{1}^{\zeta_{\zeta}} M_{2}^{\gamma_{\xi}} | M_{1}$ and $M_{2}$ are TMs, and $M_{1}$ halts on $e$ but $M_{2}$ doesn't halt on $e$ is recursively enumerable, but not recursive.

(f) ( ) The set of all primitive recursive functions is a proper subset of the set of all $1$-recursive functions.

(g) ( ) Let $A$ and $B$ be two disjoint recursively enumerable languages. If $\overline{A \cup B}$ is also recursively enumerable, then it is possible that neither $A$ nor $B$ is decidable.

(h) ( ) Let $H_{10} = \{M | M$ is a $\mathrm{TM}$ and $10 \in L(M)\}$ and $\tau_{1}$ and $\tau_{2}$ are recursive functions. If $H_{10} \leq_{\tau_{1}} L$ and $\overline{H_{10}} \leq_{\tau_{2}} L$, then $L$ is recursively enumerable but not recursive.

(i) ( ) Suppose SAT $\leq_{P} L$ and $L \in \mathbb{P}$. Then $\mathbb{P} = \mathbb{NP}$.
