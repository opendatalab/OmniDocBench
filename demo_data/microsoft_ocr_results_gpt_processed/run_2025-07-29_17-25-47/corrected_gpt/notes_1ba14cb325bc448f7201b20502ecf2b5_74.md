# (3) 西部地区农业分布

天然草场广布，以牧业为主，种植业只分布在有灌溉水源的平原、河谷和绿洲。

我国四大牧区：内蒙古牧区、新疆牧区、青海牧区、西藏牧区。

# 3. 发展农业要因地制宜

1. 宜林则林：地形陡峭，水土容易流失，但是发展林业可以保持水土。

2. 宜粮则粮：地形平坦，土壤肥沃，水源丰富，利于农作的地方，适宜种粮。

3. 宜牧则牧：降水较少，但是牧草生长良好，适宜发展畜牧业。

4. 宜渔则渔：河湖较多的地方和沿海地方，适宜发展渔业。

# 4. 我国农业面临的挑战与对策

(1) 问题：耕地减少、水土流失、自然灾害、环境污染等。

## (2) 对策：

1. 退耕还林还草，植树造林种树。

2. 科技兴农，运用现代科学技术发展优质、高产、高效农业。

3. 调整产业结构，发展多种经营业。
