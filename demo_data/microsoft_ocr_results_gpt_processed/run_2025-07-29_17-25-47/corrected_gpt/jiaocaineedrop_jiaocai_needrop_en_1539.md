# 能力秀技场

## 四、生活中的数学问题。

1. 一种圆柱形通风管，管口直径是$10cm$，长$1.2m$，做$100$根这样的通风管，至少需要铁皮多少平方米？（得数保留整平方米）

2. 一个圆柱的侧面展开后是一个边长为$15.7cm$的正方形。这个圆柱的表面积是多少平方厘米？

3. 一根长$2m$、底面半径是$4cm$的圆柱形木段，把它锯成同样长的$4$根圆柱形的木段。表面积比原来增加了多少平方厘米？

# 生活大舞台

小明把一段长$20dm$的圆柱形木头沿着底面直径锯开，表面积增加了$80dm^{2}$，原来这段圆柱形木头的表面积是多少平方分米？

例2: 做一个底面半径是$20cm$、高$50cm$的圆柱形铁皮水桶，至少需要多少平方厘米的铁皮？（得数保留整百平方厘米）

思路分析：这是求圆柱表面积在生活中的应用，求做一个圆柱形铁皮水桶需要多少平方厘米的铁皮，就是求这个圆柱形水桶的侧面积 + 1个底面积（水桶有底无盖）。

解答：$3.14 \times 20^{2} + 2 \times 3.14 \times 20 \times 50$

= $7536cm^{2}$

≈ $7600cm^{2}$

温馨提示：在实际应用中，使用的材料都要比计算得出的结果多一些，所以在取近似值时要应用“进一法”取值。
