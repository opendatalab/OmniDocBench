# 笔记十九 走向世界的中国

## 促进区域的可持续发展

### (1) 基本国情:

1. 国土辽阔, 区域差异大

2. 人口众多, 人均国民生产总值低, 为世界最大的发展中国家

### (2) 区域差异:

1. 可持续发展, 因地制宜发展经济

2. 区域合作: 加强区域联系, 互惠互利

1. 三北防护林, 东起黑龙江的县, 而至新疆的乌孜别里山. 包括13个省区的西北、华北、东北地区, 被誉为“世界生态工程之最”

### 2. 四大跨世纪工程:

| 重点工程 | 输送资源源地 | 输送目的地 |
| - | - | - |
| 南水北调 | 西线方案: 长江上游的通天河、雅砻江、大渡河 | 入黄河，解决西北的缺水问题 |
| 南水北调 | 中线方案: 汉江上游的丹江口水库 | 北京和天津 |
| 南水北调 | 东线方案: 长江下游的扬州三江营 | 河北、天津 |
| 西气东输 | 新疆塔里木的轮南油田 | 上海 |
| 南部大通道 | 怒江、澜沧江、南盘江、北盘江和红水河水电站 | 广东 |
| 中部大通道 | 金沙江干支流水电站、葛洲坝和三峡水电站 | 华东地区 |
| 北部大通道 | 黄河上游水电站和山西、蒙西地区坑口火电站 | 北京、天津、唐山 |
