解: AB ∥ CD

∴ △AOB ≌ △COD

即 CB = CD, AB=1, CD=2, AC=6
∵ CO = AO

∴ AC = AO + CO = 3AO ⇒ 3AO = 6, AO = 2
∴ S△AOB = (AB)² = (1)² = 1

S△COD = (CD)² = (2)² = 4
∴ S△COD = 4

S△AOB 与 S△BOC (不同底,同高)

底之比为 OA:OC = 1:2

S△AOB : S△BOC = 1:2 ∵ S△AOB = 1

∴ S△BOC = 2. 同理: S△AOD 与 S△COD (不同底,同高)

底之比为 OA:OC = 1:2 ∴ S△AOD : S△COD = 1:2
