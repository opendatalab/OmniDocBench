对于该题,知识点为:平行线分线段成比例和相似。

就记谁:“对应”,“交叉可互换”

1. A. 选项:
   \[
   \frac{AD}{AB} = \frac{AN}{AM}
   \]

2. B. 选项:
   \[
   \frac{BD}{MN} = \frac{CE}{AE}
   \]

3. B. 选化:
   \[
   AD = AN - AE
   \]

4. C. 正确

5. D. 选项:
   \[
   \frac{DN}{AN} = \frac{NE}{BM} = \frac{AM}{MC}
   \]

延伸:
(纯白创)

已知: \(DE \parallel BC\)
\(AN = 2NM\)

求 \(\frac{S_{ADEM}}{S_{AABC}} = R\)

看到面积 平行 比例

→直接锁定:相似

(相似比) = 面积比 ⇒ 这就是对知识点的敏感度
