# 笔记十一 从世界看中国 (“金砖四国之一)

## 边疆的疆域

### 1. 位置优越

(1) 半球位置: 东半球, 北半球.

(2) 海陆位置: 亚洲东部, 太平洋西岸

(3) 纬度位置: 大部分位于中纬度地区, 属北温带. 南部少数地区位于北回归线以南的热带, 没有寒带。

## (4) 位置优点

| | | 我国地域广大, 南北, 东西气候差异大, 为发展多种农业经济提供了有利条件. | |
| | 发展农业生产 | 经济提供了有利条件. | |
| | 对外交往与合作 | 东濒太平洋, 多优良港口, 海上交通便利, 有利于同海外各国友好往来; 西部深入亚欧大陆内部, 与多国接壤, 使我国陆上交通能够与中亚、西亚、欧洲直接往来. | |
| | 发展海洋事业 | 东临太平洋, 沿海多优良港, 便于开发各种海洋资源 | |
| | 其他 | 成就我国丰富的旅游资源, 如北国冰雪, 南国热带风光. | |

## 2. 国土面积

(1) 陆地面积: 约960万km², 差不多与整个欧洲的面积相等, 仅次于俄罗斯, 加拿大, 在世界各国中居第3位, 是世界上面积较大的国家.

### (2) 领土四至

最东: 黑龙江与乌苏里江主航道中心线的相交处 (135°E);

最西: 新疆的帕米尔高原 (73°E);

最南: 南海的南沙群岛中的曾母暗沙 (4°N);
