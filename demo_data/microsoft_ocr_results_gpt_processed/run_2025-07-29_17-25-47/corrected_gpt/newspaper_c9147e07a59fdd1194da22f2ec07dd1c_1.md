# 创新工作方法 谋求突破发展

--10省区新闻出版局负责人农家书屋建设经验谈

精挑细选 符合农民“口味”
四川省新闻出版局 王道全

![](figures/1.1)

为确保工程质量，我们把握好出版物选配这个关键，为全省农家书屋统一配备图书、报刊和电子音像制品，并按照“一书一册无复本”的原则，实现品种、数量最大化，最大限度地满足群众多样性、多层次需求，力争使选配的出版物更加适应农村经济和社会事业发展的新要求，符合农民“口味”。针对民族地区特殊情况，还将加强藏区、彝区农家书屋双语出版物的配备，确保农牧民群众看得懂、听得懂和用得上。针对灾后恢复重建的实际，将为灾区农家书屋选配一批心理抚慰、卫生防疫、灾后重建以及残疾人生活护理类的出版物，以满足不同群体的特殊阅读要求。

四川面积大，点多面广，乡村交通基础设施差，针对这一实际，通过公开招标，对配送服务实行分类采购，集中配送。几年来，我们严格按照《政府采购法》、《招投标法》的规定和财政部门的要求，委托政府采购中介代理机构，对农家书屋所需的出版物及配送服务组织实施公开招标采购，2010年，与246家中标单位签订了供货合同。

![](figures/1.2)

湖北省坚持一手抓建设，一手抓已建书屋管理和作用的发挥，全面进农家书屋工程。

武汉市江夏区和新洲区、长阳土家族自治县，黄冈市黄州区、宜城市、仙桃市等地发挥县级图书馆专业优势，通过举办培训班等形式，大力加强管理员教育培训，有的还颁发农家书屋管理员资格证书，强化了管理员的管理服务意识和水平。武汉市部分区财政还为每个管理员按每年3000元标准给予补助。

各地通过免费向村民发放读书卡和主要藏书目录及农家书屋宣传单，粉刷标语，农村电影巡回放映推介农家书屋等形式，使农家书屋家喻户晓，营造读书氛围。利川市团堡镇樱桃井村农家书屋，根据村民深居大山、居住分散特点，由各村民小组组长代为村民借书，既宣传了农家书屋，也方便了群众。

开展读书活动。全省各地依托农家书屋开展主题鲜明、形式多样、与农民群众生产生活息息相关的读书用书活动，如读书征文、讲演比赛、知识竞赛、科技培训、普法宣传等，引导村民到农家书屋读书用书，增强农家书屋的吸引力，努力培养农民群众阅读兴趣，努力使农家书屋成为农村学生的“第二课堂”。

![](figures/1.3)

团组织参与农家书屋管理，既有利于加强基层团组织建设，推进青年文化建设，增强团组织的凝聚力，又能使农家书屋借助共青团的组织体系，建立起有一定组织形式的农家书屋管理员队伍，并依托农家书屋开展有组织的、丰富多彩的活动。

2009年1月，辽宁省局与团省委共同部署和安排，14个市的团组织与新闻出版局签订了合作协议，明确了农家书屋建设以出版局为主，后期管理特别是农家书屋管理员的选配以团组织为主，配齐配强农家书屋管理人员，建立健全管理制度，开展农家书屋管理员的培训，依托农家书屋开展读书日、征文比赛、农业实用技术培训等特色活动。抽样调查表明，全省86%的管理员是由团干部、团员青年和青年创业致富带头人担任的，各项工作得到了很好的落实。

共青团参与农家书屋管理工作，不仅使各级团组织有了新的活动载体和工作阵地，提升了服务青年的能力，也使我省农家书屋逐步实现“有专人负责的管理，有健全完善的制度、有专业技能的培训，有特色丰富的活动，有创新模式和机制”的“五有”目标有了组织上的保障。

![](figures/1.4)

我局先后制定了《山西省农家书屋工程建设资金管理办法》、《山西省农家书屋建设管理办法》等示范性规章制度，根据实际情况增补删减，做到管理规范、各具特色；高度重视工程质量管理，严格落实验收标准，从组织领导、规划选址、图书配送、设施建设、内部管理等方面，对农家书屋建设工作进行全面考核，凡不达标的书屋全部责令整改；把建好管理员队伍，不断提高管理员素质作为工作重点来抓，倡导大学生村官担任农家书屋管理员。

对各地涌现出来的典型，注重及时总结经验，不断进行推广。选择晋中市榆次区为重点示范区，进行了农家书屋和图书馆联动运行的试点工作，把农家书屋作为榆次区图书馆的分馆，将农家书屋的图书作为馆藏图书进行管理，解决了图书流动问题。晋城市则总结出利用乡镇综合文化站、村级组织建设等资源进行整合的金村镇金村的典型等四种模式在全省推广，引导各地因地制宜，不断创新，推动农家书屋的建设。

![](figures/1.5)

将继续加强与财政部门的沟通协调，争取更多的资金用于农家书屋建设。省财政在过去每年投入1000万元农家书屋建设专项资金，实行以奖代补的基础上，2011年将大幅度提高省级财政支持力度，专项资金投入要翻两番，并以此为带动，增加市、县、乡各级财政对农家书屋工程建设的投入。

为动员市、县增加投入，2010年底，省局所有局长到市、县去争取资金支持。在发挥各级财政建设主渠道的同时，多措并举、创新形式，广开思路，积极发动和鼓励社会各界捐建农家书屋工程。

同时注意发挥好三个渠道的作用：一是与省文明办、省委机关工委联合倡议开展精神文明单位带头捐建农家书屋活动；二是开展“我为家乡捐建农家书屋活动”，广泛动员机关、社会团体、全事业单位、学校等社会各界积极捐建农家书屋；三是主动与省各民主党派和工商联加强联系，发挥他们与经济界、企业界联系广泛的优势，动员更多的民主党派成员投入到积极捐建农家书屋的行列中来。

![](figures/1.6)

紧密结合实际，突出浙江特色，务求取得实效、实现长效。

一是加强资源整合，将文化、民政、计生、精神文明建设等部门建设的农村阅览室等具有农家书屋功能的文化设施，统一纳入农家书屋体系，节约建设成本，提高建设成效。

二是发达地区先行，经济发达地区的农家书屋建设成效明显，水平较高，目前杭州、宁波、嘉兴和舟山四个地区已基本实现了农家书屋全覆盖。

三是创新送书下乡，省政府连续5年将送书下乡活动列入“为民办实事”的十件实事之中，每年向农村送200万册图书，我局连续5年组织实施“服务‘三农’重点出版物出版工程”等。

四是开展书屋活动，充分发挥农家书屋的作用，不少地方在书屋里开设“书香书场”，办阅读节、举行文体活动，有些地区的农家书屋还定期开设培训课程，从国家政策、乡风民俗、计划生育、外来务工人员管理、少先队建设、老年人保健等多方面对农民进行有针对性地培训教育，深受农民群众的欢迎。

出台政策高起点，建设质量高标准。在新闻出版总署下发文件的基础上，结合实际，先后下发关于转发新闻出版总署农家书屋工程建设领导小组《关于各地农家书屋验收工作应当注意问题的通知》等文件，对全区农家书屋整体建设计划及完成时间提出了更高的要求，即农家书屋建设必须达到“十有”、“四统一”标准建设。

加强有效有序的管理，制定了农家书屋财产管理办法，要求各市所有农家书屋必须建立农家书屋财产登记合账，确定专人对农家书屋大到书柜、小到每本书、记录簿等都要按照进书时间、金额登记入账，并给管理员定职责，明确一名专职管理员来管理农家书屋的日常运作。

对建成的农家书屋，做好“六个结合”：与新农村建设结合，与文明行为养成结合、与科技培训结合，与法律普及结合，与婚育新风结合，与和谐邻里关系结合，同时积极规划、策划一些内容积极向上、富有吸引力的活动，来引导农民群众走进书屋、走近书桌。

河北省有49347个行政村，农家书屋工程总投资需近10亿元，建设任务居全国第二位。在这样的形势下，我们组织了“帮扶联系活动”，广泛发动党政部门、高校、企业和爱心人士参与，形成了全社会关注、支持、参与农家书屋工程建设的良好局面。活动自开展以来，我有6341个单位和个人建立农家书屋帮扶联系点6440个，帮扶农家书屋4440个，举办各类讲座270多场（次），受益农户500多万。

2011年是实施“十二五”规划的开局之年，河北省委、省政府对加强农村文化建设提出了更高的标准和要求。在2010年足额落实地方配套资金2.2亿元确保农家书屋工程建设进度的基础上，今年，河北省政府又设立了“河北出版政府奖”和新闻出版产业引导资金，并拿出3000万元作为农家书屋工程的奖励资金和工作经费，根据进度对各县、市每新建一个书屋给予1200元-2400元的标准奖励。各级政府和有关部门下达建设规划，签订责任状，加强工作督导，严格目标考核，确保经济条件居中上游的50个县、市、区在2011年实现全覆盖；经济条件居中下游的115个县、市达到70%。8月底前，再新建农家书屋13890个，确保到2012年实现农家书屋行政村全覆盖。

## “两个民生工程”一起做 江西省新闻出版局 黄鹤

就业“两个民生工程”一起做，由县残联和新闻出版部门共同组织残疾人管理员培训，培训计划和经费列入当地劳动就业保障部门的农民工培训规划。县残联负责培训的组织与保障，新闻出版局负责培训内容、师资和考核，并由这两家单位共同核发农家书屋管理员上岗证。

2010年11月，我局农家书屋办公室与省残联组成联合调研组，对农家书屋残疾人管理员工作绩效进行了考评。调研显示：残疾人管理员对加强农家书屋的管理与使用起到了很好的作用，既保证了开放时间，方便了村民们借阅图书，又使农家书屋资产管理得到了加强，图书流失现象大大减少。

积极探索解决各种突出问题的途径与措施。针对当前书屋运行成本较高等突出问题，积极开动脑筋，积极推广“四公开”制度，要求所有农家书屋将开放时间、管理员姓名、联系方式、监督方式在书屋外墙显著位置予以公布，大力加强日常管理等措施。通过农村低保途径以及市场化途径解决书屋管理成本和管理人员报酬。

农家书屋的受惠范围进一步得到扩大。去年以来，针对我省少数民族数量较多，少数民族信教群众较多，民族文化活动场所文化基础设施薄弱的特点，联合省宗教局进行广泛调研，尝试将个别行政村的农家书屋建在少数民族宗教文化活动场所。
