# 10.相互代词:表示相互关系的词叫相互代词。

each other,one another是相互代词,译为“互相”,可以通用。each
other表示两者之间,而one another表示许多人之间。它们有所有格形式:
each other's, one another's.

如: We must help each other when we are in trouble.
(我们身处困境时要互相帮助.)

They sat there without talking to one another/each other.
(他们坐在那儿,互相都不说话。)
