# 盈利预测

公司作为一家特色品牌中药企业,多个核心品种处于增长
期,并通过并购进入医美领域,探索寻求新的业绩增量,预
测公司2022-2024年收入分别为8.66、9.55、10.59亿元,
EPS分别为0.85、1.07、1.27元,当前股价对应PE分别为
17.2、13.7、11.5倍,给予“推荐”投资评级。
