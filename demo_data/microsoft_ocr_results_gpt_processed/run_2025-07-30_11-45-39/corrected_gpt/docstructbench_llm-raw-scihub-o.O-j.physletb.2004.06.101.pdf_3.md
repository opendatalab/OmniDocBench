For consistency, the time derivative of the constraints of (10) must vanish and hence they must have vanishing
Poisson bracket with $H$ . Using the fundamental Poisson brackets

$$\left[ U \left( x \right) , \Pi ^ { U } \left( y \right) \right] = \delta \left( x - y \right) ,$$
(13)

etc., we find that the primary constraints of (10) imply the secondary constraints

$$\left( \Sigma , \Sigma _ { i } \right) = \left( - \partial _ { k } \Pi _ { k } ^ { V } , \varepsilon ^ { i j k } \partial _ { j } \left( \Pi _ { k } ^ { B } - m V _ { k } \right) - \mu ^ { 2 } B _ { i } \right) .$$
(14)

If $\mu ^ { 2 } = 0$ (the Cremmer-Scherk model Lagrangian [1]), the constraints of (14) would become reducible as then
$\partial _ { i } \Sigma _ { i } = 0$ and only the transverse portions of $\Sigma _ { i }$ are constraints. Furthermore, with $\mu ^ { 2 } \neq 0$ , the requirement $\dot { \Sigma } _ { i } = 0$
leads to a tertiary constraint

$$T _ { k } \equiv \mu ^ { 2 } \Pi _ { k } ^ { B } = 0$$
(15)

with $\Sigma _ { i }$ and $T _ { k }$ constituting second class constraints as

$$\left[ T _ { k } \left( x \right) , \Sigma _ { i } \left( y \right) \right] = \mu ^ { 4 } \delta _ { i k } \delta \left( x - y \right) .$$
(16)

All other constraints are first class and no further constraints need to be imposed for consistency. There are
consequently five first class constraints $\left( \Phi ^ { U } \right.$ , $\Phi _ { k } ^ { A }$ and $\left. \Sigma \right)$ and six second class constraints $\left( \Sigma _ { i } \mathrm { and } T _ { k } \right)$ . The con-
straints $\Phi ^ { U }$ $\Sigma$ correspond to the usual gauge transformations $\delta W _ { 0 } = \partial _ { 0 } \Omega$ , $\delta W _ { i } = \partial _ { i } \Omega$ associated with a gauge
field $W _ { \mu } ,$ while $\Phi _ { k } ^ { A }$ is associated with the fact that in (12) $A _ { k }$ acts merely as a Lagrange multiplier (i.e., it is not
dynamical) and hence its value is completely arbitrary. Suitable gauge conditions associated with the first class
constraints are

$$\left( \gamma ^ { U } , \gamma _ { k } ^ { A } , \gamma ^ { V } \right) = \left( U , A _ { k } , \partial _ { k } V _ { k } \right) = 0 .$$
(17)

From (10), (14), (15) and (17) it is evident that the only dynamical degrees of freedom are

$$V _ { i } ^ { T } \equiv \left( \delta _ { i j } - \partial _ { i } \partial _ { j } / \partial ^ { 2 } \right) V _ { j } .$$
(18)

We can verify this directly by explicitly eliminating the non-physical degrees of freedom in (4). First, one
decomposes $V _ { k }$ , $A _ { k }$ and $B _ { k }$ into transverse $\left( T \right)$ and longitudinal $\left( L \right)$ parts where

$$\mathrm { V } \times \mathrm { V } ^ { L } \equiv 0 \equiv \mathrm { V } \cdot \mathrm { V } ^ { T } ,$$
(19)

etc., (4) now becomes

$$2 L = \left( \dot { \mathrm { B } } ^ { L } \right) ^ { 2 } - \left( \mathrm { V } \cdot \mathrm { B } ^ { L } \right) ^ { 2 } + \left[ \dot { \mathrm { B } } ^ { T } - \mathrm { V } \times \mathrm { A } ^ { T } \right] ^ { 2 } + \left( \dot { \mathrm { V } } ^ { T } \right) ^ { 2 } - \left( \mathrm { V } \times \mathrm { V } ^ { T } \right) ^ { 2 } + \left[ \dot { \mathrm { V } } ^ { L } - \nabla U \right] ^ { 2 }$$
$$+ 2 m \left[ \mathrm { V } ^ { T } \cdot \left( \mathrm { V } \times \mathrm { A } ^ { T } \right) + \mathrm { B } ^ { L } \cdot \dot { \mathrm { V } } ^ { L } + \mathrm { B } ^ { T } \cdot \dot { \mathrm { V } } ^ { T } - \mathrm { B } ^ { L } \cdot \nabla U \right] + 2 \mu ^ { 2 } \left[ \mathrm { A } ^ { T } \cdot \mathrm { B } ^ { T } + \mathrm { A } ^ { L } \cdot \mathrm { B } ^ { L } \right] .$$
(20)

The equations of motion for $\mathrm { A } ^ { L }$ and $U$ , respectively, imply that

$$\mathrm { B } ^ { L } = 0 = \dot { \mathrm { V } } ^ { L } - \mathrm { V } U ,$$
(21)

reducing (20) to

$$2 L = \left( \dot { \mathrm { V } } ^ { T } \right) ^ { 2 } - \left( \mathrm { V } \times \mathrm { V } ^ { T } \right) ^ { 2 } + \left[ \dot { \mathrm { B } } ^ { T } - \nabla \times \mathrm { A } ^ { T } \right] ^ { 2 } + 2 m \mathrm { V } ^ { T } \cdot \left( \nabla \times \mathrm { A } ^ { T } \right) + 2 m \mathrm { B } ^ { T } \cdot \dot { \mathrm { V } } ^ { T } + 2 \mu ^ { 2 } \mathrm { A } ^ { T } \cdot \mathrm { B } ^ { T } .$$
(22)

Since

$$\mathrm { A } ^ { T } \cdot \mathrm { B } ^ { T } = - \left( \nabla \times \mathrm { A } ^ { T } \right) \cdot \left( \nabla ^ { 2 } \right) ^ { - 1 } \left( \nabla \times \mathrm { B } ^ { T } \right) ,$$
(23)

we can eliminate $\mathrm { V } \times \mathrm { A } ^ { T }$ from (22) to obtain

$$\nabla \times \mathrm { A } ^ { T } = \dot { \mathrm { B } } ^ { T } - m \mathrm { V } ^ { T } + \mu ^ { 2 } \left( \nabla ^ { 2 } \right) ^ { - 1 } \left( \nabla \times \mathrm { B } ^ { T } \right) .$$
(24)
