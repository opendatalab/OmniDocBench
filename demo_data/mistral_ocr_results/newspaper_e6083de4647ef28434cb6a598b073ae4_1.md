# 全国政协举行新年茶话会

**2011年1月3日** 夏腊中 **表态褒奖** 举办月 **日电(4)@张宏堂** 吴晶晶(新年伊始,万象更新,中国人民政府将商业议会部署员会4日上午在全国政协礼堂举行新年茶话会,现场休,吴明信、温家宝、贾庆林、李长春、习近平、李友谊、莫国强、周永强等党和国家领导人同各民主党派中央、全国工商联领导人和无党派人士代表,中央和国家机关有关方面负责人以及首都各族各界人士代表欢聚一堂,员次2011年元旦。

茶话会上,胡锦涛等领导同志来到各界人士中间,亲切地同大家握手交谈,自我问题、文艺工作者表演了歌舞,由艺等精彩节目。

# 我国食用菌年产值预计1500亿元

**本报讯(记者 徐发升)**日前,记者从“全国第三项食用菌工厂在生产达标”上获悉,预计2000年全国食用菌总产量达2800万吨,产值1800亿元,1+10月估计前工计记美元。

近几年,随着国家对发展特色农业和设施农业的作为提高不断加大,食用菌传统的季节性生产模式已经被打破,工厂化生产得以迅速发展。据中国食用菌协会统计,目前全国在建和已建成投产的工业企业已达成440家,共总产量达3000年的40万吨上升到48万吨,增幅达42.8%,食用菌工厂化生产的总种也比较限的金针菌、风卷菌、扩展剂含酸菌、蟹纲菌、新鲜菌、患子菌等十几个品种,致学家介绍,我国的食用菌工厂化生产技术和管理已达到国际先进水平。

# 广东加快抓升

**现代农业物质装备水平**

**本报讯(记者 陈婕** 通讯员 **黄玉海** 日前,广东省政府在佛山市召开全省提升现代农业物质技术装备水平工作会议。记者在会上了解到,“东亚先后实施农村机电排灌建设、农田购粪坑建设和人工消磨建设等议事,推动现代农业物质技术装备特别是农业机械化快速发展,按照管理、科研推广和社会可服务”三大体系下不断完善,水稻生产全程机械化、园艺生产机械化、畜牧养殖机械化和地产添加工机械化“四大工程”建设卓有成效。农业物质技术装备总量快速增长,应用水平快速提升,创新能力显著增强,社会化服务领域不断拓宽。

广东省省长黄华华在会上指出,当前80年在一个时期,“东南亚南越升现代农业物质技术装备水平,作业水平,产业化水平和装备制造水平,为建设现代农业服务提供有力支撑。力争到2015年,全省农业装备总量大幅增加,农机总动力达到2700万千瓦,水稻生产全程机械化快的推进,水稻耕种收拾合利,畸化水平达到65%,年产养殖机械覆盖率达到80%以上。

# 河北将建周转房

**农村教师免费住**

**本报讯(记者 郝建峰)**民间北省中北酷酷西南第和发展规划纲要(2010-2020年)针对农家建筑楼1年建设,800到2012年,同北省将组织实施12个重大项目,其中农村中小早教师住房建设工程的实施将有望在“零负担”的基础上解决农村教师住房困难问题。

由于住房改革推行后,地方财政对农村中小早教师住房建设的资金补助要虚弱切断,教师住房问题日益严重。农村酷酷发展后,依靠群众集资建设教师住房的办法也不可能再推行,购买商品房的方式对于教师阶层来说负担过重。

回顾职地此报出,按照“政府建设,学校管理,教师无偿入住”的原则,由各级政协安排专项资金,支持农村中小早教师周转房建设。鼓励各地利用安全可用的闲置机房建设新的周转房。通过制定土地、规费减免等优惠政策,支持教师住房建设。对符合住房保障条件的,各地配套对照其纳入保障范围。

# (快乐农庄)网游在海口开通

**本报讯(记者 梁龙** 通讯员 **吴高)**近日,国内首款以农业科技知识普及功能为基础,并结合展示海南热带特色农业魅力的真实模拟农业生产进场免予农务网游投(实农庄)给海南省海口市正式开通。

电子农务网游投(实农庄)紧密结合了海南热带特色农业的实际生产情况,将农作物刚出销的区,增产增收转嫁的生产技术有机融入游场过程,阻碍了传统科普的检测评解,可以大大调动农民回家学习农业科技知识的积极性,引导农民回家科学种植、勤劳致富,达成氧教于乐的目的。同时,该款游戏还可助同龄幼儿摄像机,将海南热带特色农业的生产过程,风土人情,自然风貌等生动形象地介绍如何家,激发民间消费的产品和实地旅游的欲望,促进海南热带特色休闲观光农业的发展,提升国际旅游高魅力。
