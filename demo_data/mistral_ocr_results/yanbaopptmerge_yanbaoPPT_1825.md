# 已知三角形的两边及其夹 

数学活动 2 角, 求作三角形

$$
\begin{aligned}
& \text { E } \\
& A B=\text { c, } \angle A B C=\angle a \\
& \text { a } \\
& \text { b }
\end{aligned}
$$

$$
\begin{aligned}
& \text { D } \\
& \text { 作法 } \\
& \text { M } \\
& \text { (1) 作 } \angle M B N=\angle a \\
& \text { (2) 在射线 B } M \text { 上截取 } B C= \\
& \text { a, 在射线 B } N \text { 上截取 } B A= \\
& \text { b (3) 连接 } A C
\end{aligned}
$$

则 $\triangle A B C$ 为所求作的三角形
