# 3. 我国木桶种植业 

费型地区：大型平高，铁立三角型等
公有民故因素：
(1)自然因素：热量充足，的热周期；三角型地热平坦，工需把关；河网密布，大需充足。

(2) 社会经济因素：和利汗水平高，交路距利；市街广阔；工业化较发达；科技发达；国际政策扶持。

不足：人多地少，各色风，景的造成的洪涝灾害影响；国的范围。

## 4. 高农业

公有民故因素：
(1) 自然因素：草地面积广阔；没有大型良河动物；水路地区是考得水平色有的于铁草生长（得水从东向西递低）。

不足：全境干旱，得水少；草地强化；耐潮化加的；冬考景良富，影响影响；损灾、蝗灾严重。
