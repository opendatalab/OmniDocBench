# 三部门再次公布举报电话 严查虚假报道

新华社北京12月31日电 为深入推动
全国新闻战线开展“杜绝虚假报道、增强社
会责任,加强新闻职业道德建设”专项教育
活动,国家广电总局,新闻出版总署、中国记
协近日再次向社会公布举报电话。三部门
有关负责人表示,有关单位和社会各界群众
发现媒体刊载虚假报道的,可拨打相关举报
电话,三部门将对举报内容进行认真核查。
一经核实确为虚假报道的,将严肃查处,绝
不姑息;造成恶劣社会影响和严重后果的,
将依法依规追究相关责任人和新闻单位责
任。各新闻单位和新闻从业人员要加强自
律,自觉接受社会监督:对出现虚假报道问
题的,要做到不回避,不护短,勇于承认错
误,及时更正效歉,挽回不良影响。要通过
新闻界的自律和社会化的监督网络构筑防
治虚假报道的“防火墙”,确保新闻报道客观
真实,切实维护新闻真实性和媒体公信力。
维护新闻工作者良好社会形象。

据悉,各省,自治区,直辖市有关部门
和单位也将陆续公布举报电话。

## 2011年中国工程院 增选60名院士

本报讯(记者 张丛 实习生 胡月)记者
从中国工程院获悉,该院院土增选工作1月
1日正式自动,增选名额为60名。最新修改
完善的《中国工程院院士增选实施办法》中
对增选院士的学术道德做了更严格的规定,
除了增加候选人材料在“基层单位和内网公
示”,也首次用确对违规行为进行直接处罚。

此次增选中开始实施的《院士增选办
法》规定,被提名人报送的材料”一旦发现材
科不实,弄虚作假,经审核后以学风和道德
问题作出处理,终止该被提名人的评审与选
举”;提名单位报送的候选人材料”一旦发现
材料有严重不实,弄虚作假等情况,经审查
审核,将终止该候选人的评审与选举,取消
下一次提名的资格”。


## 中国文联、美协赴海南 “送欢乐、下基层”

据新华社海口12月31日电(记者 夏冠
男)12月29日到2011年1月3日,由中国文
联,中国美术家协会组织的“送欢乐,下基
层艺术家小组赴海南慰问。其间,16位艺
术家组成的小组将深入到海南农村,社区等
基层单位和灾后重建第一线,看望受灾群众
和黎族群众,给他们送去新春的祝福。

据中国美术家协会分党组书记吴长江
介绍,“送欢乐,下基层”小组一行先后来到
海口市龙塘镇信南村慰同在2010年暴雨洪
沸灾害中受灾的群众,为150多户灾民发放
了慰问金;深人五指山黎村,体验文明生态
村建设,为农民书写春联和为黎族群众画
画像,把文化艺术送到田间村头。”送欢乐,
下基层”小组还到琼海市阳江镇歌问红色
娘子军老战士,为她们造像,给她们送去新
春的祝福,转达党和政府对灾区人民的深
切关爱,表达全国广大美术家对灾区人民
的深情厚谊、对革命老前辈的崇敬和爱戴。

12月31日,中国美术家协会与中共海
南省委宣传部还共同举办了“魅力海南岛
--中国画百家百景”大型美术创作,展觉
活动.100余名中国当代中国画老,中、青美
术名家用绘画这一独特的艺术形式展现“琼
州百景”。

据介绍,艺术家们还将到企业,山区,
海边等地采风,写生,用作品表现海南人民
灾后重建的成果和人民群众昂扬向上的精
神风貌。

# 乌拉特草原遭暴风雪袭击 近万名受灾牧民得到救助

新华社呼和浩特12月31日电
[记者 李云平]记者31日从内蒙古自
治区巴彦淖尔市政府获恶.12月28日
以来,内蒙古自治区乌拉特草原遭暴
风雪袭击,近万名牧民受灾,部分牲
畜被冻死 截至目前.受灾群众得到
及时教助,生产生活逐新恢复

陈金柱是乌拉特中旗川井苏
木呼格吉乐嘎查牧民,家里养着
620只羊。12月28日,他在放羊过
程中遭遇暴风雪天气,羊群全部走
失。第二天清晨,他寻找羊群时发
现已有560多只羊被冻死。陈金柱
说:“我找了一天就找见40多只羊,
当地政府派人帮我又找到10多只

羊,剩余的羊都让白雪覆盖了。”

在这次暴风雪天气过程中,乌
拉特中旗川井苏木和巴音乌兰苏
木均遭受暴风雪袭击,受灾牧民
1184户,4893人,受灾牲畜35.5万
头(只),死亡牲畜4394头(只)。灾
情发生后,当地政府及时组织人员
帮助牧民寻找羊群,抢救活畜,为
他们送去米面等救济物品,详细灾
情还在进一步统计中。

与此同时,乌拉特后旗5个苏
木镇也遭受暴风雪袭击,受灾牧民
1384户,4928人,受灾牲畜24.8万
头(只),死亡牲畜2292头(只)。灾
情发生后,当地政府及时自动处置

突发暴风雪灾害应急预案,同时开
展核查灾情,发放救灾物资等抗灾
救援工作,力争把灾情带来的损失
降最低限度。

从12月28日夜间开始,乌拉
特中旗,乌拉特后旗公安、交警、边
防派出所等緊急行动,授救被暴风
雪围困的群众。截至目前.40多名
被困牧民成功获救,无一人伤亡。

据介绍,针对近万名牧民受灾,牲
畜饲草料缺口较大等实际情况.当地
政府正在妥善安排灾区群众生活,确
保受灾群众有饭吃,有衣穿,有取暖燃
料,有病得到及时医治,紧急调运油,
料等储备物资,帮助收民抗火自救。


# 汽车下乡补贴政策 到期停止执行

新华社北京12月31日电 财
政部31日公布了2011年继续实施
和停止执行的部分財稅优惠政策,
明确对部分小型散利企业的所得税
优惠政策、推广节能汽车和新能源
汽车的财政补贴政策等将继续实
行。与此同时,汽车下乡和汽车以
旧換新补貼政策于2010年12月31
日执行完毕。

财政部明确,继续实行结构性
减税是2011年积极財政政策的重要
内容之一。继续对部分小型微利企
业实施所得税优惠政策,支持中小企
业发展。落实好各项税费减免政策,
促进产业结构开级和服务业发展。
实施有利于节能成排、环境保护和增
加就业的税收优惠政策。继续清理

規范行政事业性收费和政府性基金
项目。另外,为加快汽车产业结构调
整升级,促进节能减排和资源循环利
用,2011年继续采取财政补贴政策,
支持全面推广节能汽车和加大新能
源汽车示范推广力度.继续实施老旧
汽车报废更新补贴政策。

此外,财政部表示,汽车下乡和
汽车以旧换新补贴政策于2010年
12月31日执行完毕。经国务院批
准,近日,财政部会同有关部门印发
了(关于汽车下乡政策到期后停止
执行等有关问题的通知X关于汽车
以旧换新政策到期后停止执行等有
关同题的通知》,明确了到期停止执
行等有关政策并对做好后续工作提
出了具体要求。


# 做一个有道德的人

各地各部门不断开创未成年人思想道德建设新局面

(记者 李菲)面向广大未成年人深
人开展爱国主义和革命传统教育,
引导未成年人用社会主义核心价值
体系武装头脑,逐步形成覆盖城乡
的未成年人心理健康教育网络··…·

作视讯会议结束后,各地各部门及
时召开会议传达学习会议精神.研
究制定具体措施,认真抓好贯彻落
实,不断开创未成年人思想道德建
设工作新局面。

江苏省要求各地各部门进一步抓
好重点任务落实,统筹推进社会主义
核心价值体系学习教育,“三结合”教
育网络建设、社会文化环境争化、优秀
文化产品创作生产等各方面工作:抓
住中国共产党成立90周年,辛亥革命

人深人开展爱国主义和革命传统教
育。进一步提高工作科学化水平,注
重规划布局,各方面联手,形成强大工

好、未成年人欢迎的实事好事。

系学习教育作为第一位的任务,贯穿
于未成年人成长全过程和国民教育

未成年人的精神世界。深入开展“做

设工作的领导,把未成年人工作作为

设作为宣传文化工作的重点来抓。
要求全市宣传文化战线把未成年人
思想道德建设作为战略工程,基础
工程,民心工程和希望工程,纳人宣
传文化工作总体安排,一起谋划,一
起部署,一起落实,一起考核。在工
作中始终围绕社会主义核心价值体
系建设开展未成年人思想道德教
青。进一步完善机制,形成未成年
人思想道德建设工作的强大合力。
