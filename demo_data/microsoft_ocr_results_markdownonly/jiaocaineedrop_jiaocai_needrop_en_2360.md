# 四、解决问题。(34分)


## 1.杭州某书店5月份每周的销售情况如下表。(8分)


| 周数 | 第一周 | 第二周 | 第三周 | 第四周 |
| - | - | - | - | - |
| 本数 | 2034 | 2988 | 1982 | 4088 |


(1)该书店每一周销售的书各接近几千本?(4分)

第一周约(2000)本
第二周约(3000)本

第三周约(2000)本
第四周约(4000)本

(2)把该书店5月份这四周销售的图书本数按从少到多的顺序排列。(2分)
1982<2034<2988<4088

(3)卖出图书的数量最多的一周比最少的一周大约多多少本书?(2分)
4000-2000=2000(本)

2.体育广场共有2400个座位。有三所学校各派出七百多名学生代表来参加演讲比
赛,这些座位够吗?(4分)
把七百多估成800。
800+800+800=2400(名)
2400=2400 够。
