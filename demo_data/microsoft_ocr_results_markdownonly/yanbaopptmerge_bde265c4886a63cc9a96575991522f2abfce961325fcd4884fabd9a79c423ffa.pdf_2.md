# 1.A股市场走势综述

周一(03月27日)A股市场先抑后扬、宽幅震荡整理,受到恒生指数显著下行的影响,早
盘两市股指小幅低开后逐级震荡回落,盘中房地产、酿酒、通信以及金融等权重股轮番走低,
沪指盘中在60日均线3230点附近获得支撑,午后随着新能源、医疗服务、软件开发以及互联
网等热门行业轮番上扬,带动股指企稳回升,沪指全天基本呈现宽幅震荡的运行特征。创业板
市场周一震荡上扬,成指全天表现强于主板市场。上证综指收市报3251.40点,跌0.44%,深
证成指收市报11647.94点,涨0.12%,科创50指数下跌0.33%,创业板指上涨1.17%。深沪两
市全日共成交11400亿元,较前一交易日有所增加。

从盘中热点来看,两市超过五成个股下跌,其中风电设备、电子化学品、医疗服务、软件
开发以及互联网服务等行业涨幅居前;房地产服务、房地产开发、水泥建材、装修建材以及铁
路公路等行业跌幅居前。光伏设备、医疗器械、医疗服务、电源设备以及风电设备等行业资金
净流入居前;半导体、互联网服务、通信服务、计算机以及软件开发等行业资金净流出居前。
