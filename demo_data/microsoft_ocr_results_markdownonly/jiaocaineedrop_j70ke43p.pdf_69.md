### (1)积化和差公式

$$\sin A \cos B = \frac { 1 } { 2 } \left[ \sin \left( A + B \right) + \sin \left( A - B \right) \right] ; \cos A \sin B = \frac { 1 } { 2 } \left[ \sin \left( A + B \right) - \sin \left( A - B \right) \right]$$
$$\cos A \cos B = \frac { 1 } { 2 } \left[ \cos \left( A + B \right) + \cos \left( A - B \right) \right] ; \sin A \sin B = \frac { 1 } { 2 } \left[ \cos \left( A - B \right) - \cos \left( A + B \right) \right]$$


### (2)和差化积公式

$$\sin A + \sin B = 2 \sin \frac { A + B } { 2 } \cos \frac { A - B } { 2 } ; \sin A - \sin B = 2 \cos \frac { A + B } { 2 } \sin \frac { A - B } { 2 }$$
$$\cos A + \cos B = 2 \cos \frac { A + B } { 2 } \cos \frac { A - B } { 2 } ; \cos A - \cos B = - 2 \sin \frac { A + B } { 2 } \sin \frac { A - B } { 2 }$$
