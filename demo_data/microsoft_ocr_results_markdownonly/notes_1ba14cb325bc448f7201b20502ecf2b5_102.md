# 2.沙尘最现象

我国典型地区:西北.毕北地区

产生的原因:

(1)自然桑因:快行冷锋天气影响:气候干早,降水少:春季大风天数多,地表指
被稀少等

(2)人为原因:过度放牧;过度撫采:过度开垦

治理措施,

制定草场保护的法律,法规,加强管理,控制載高量:营造“三北防护林建
设;退耕还林,还牧;建设人工草场;推广轮妆;禁止采戊发莱等


# 3.地震

我国典型地区:东部沿海:西南,西北地区

形成原因:

位于亚欧板块和太平洋板块,印度洋板块的交界处,地壳活动剧烈。

造成重大人员和财平损失的原因可能有:

震级大,破坏性大:霞中附近城市分币多,人口集中;浅源地震发生的时间可

能在夜间:语发其他灾害争


## 减轻灾害的措施:

积极开展防灾.减灾的宣传教育.提高公众的环保和减灾意识:建立灾害
监测预报体系;加强地质灾害的管理,建立健全减灾工作的政策法规体系;提
高建筑物的抗震强度;植树造林,建立防护林体系:加强国际合作等.
